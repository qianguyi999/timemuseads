# TimeMuse Static Website

Static HTML website for `timemuseads.com`.

## Pages

- `index.html`
- `advertising.html`
- `online-advertising.html`
- `business-info.html`
- `exhibitions.html`

## Cloudflare Pages Settings

- Framework preset: None
- Build command: leave empty
- Build output directory: `/`
- Production branch: `main`

## Update Flow

1. Edit the HTML files.
2. Commit or upload changes to GitHub.
3. Cloudflare Pages deploys the new version automatically.
4. Use Cloudflare deployment history to roll back if needed.
